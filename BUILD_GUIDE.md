# دليل بناء ملف EXE
# Building EXE Guide

## المتطلبات:
- Python 3.8 مثبت على النظام
- جميع المتطلبات من requirements.txt

## طريقة البناء السريعة:

### الطريقة الأولى (موصى بها):
1. افتح PowerShell أو CMD في مجلد المشروع
2. قم بتشغيل: `build_exe.bat`
3. انتظر حتى ينتهي البناء (قد يستغرق 3-5 دقائق)
4. ستجد الملف في مجلد `dist/SamirPOS.exe`

### الطريقة الثانية (يدوياً):
```bash
# 1. تثبيت PyInstaller
py -3.8 -m pip install pyinstaller

# 2. بناء ملف EXE
py -3.8 -m PyInstaller build_exe.spec --clean --noconfirm
```

## الملفات المضمنة في EXE:
- ✅ جميع ملفات Python (controllers, database, ui, utils, views)
- ✅ ملفات الإعدادات (config.py)
- ✅ المكتبات المطلوبة (customtkinter, PIL, sqlite3)
- ✅ جميع التبعيات

## بعد البناء:
1. ملف EXE موجود في: `dist/SamirPOS.exe`
2. يمكنك نقله لأي مكان على الجهاز
3. التشغيل الأول سيُنشئ قاعدة البيانات تلقائياً
4. **مهم**: لا تحذف ملف `pos_system.db` بعد إنشائه!

## الحجم المتوقع:
- حجم EXE النهائي: ~80-100 MB تقريباً
- يشمل Python runtime كاملاً + جميع المكتبات

## معلومات التشغيل:
- المستخدم الافتراضي: `admin`
- كلمة المرور: `admin123456`
- مستخدم البائع: `cashier` / `cashier123`

## استكشاف الأخطاء:

### إذا ظهرت رسالة "Python 3.8 غير موجود":
```bash
# تأكد من تثبيت Python 3.8
py -3.8 --version
```

### إذا فشل البناء:
1. تأكد من تثبيت المتطلبات:
```bash
py -3.8 -m pip install -r requirements.txt
```

2. حاول حذف المجلدات القديمة:
```bash
rmdir /s /q build
rmdir /s /q dist
```

3. أعد المحاولة

### إذا لم يعمل EXE:
1. تأكد من وجود صلاحيات التشغيل
2. شغّله كمسؤول (Run as Administrator) في المرة الأولى
3. تحقق من برنامج مكافحة الفيروسات (قد يحجبه)

## ملاحظات:
- ✅ يعمل على Windows 7 والأحدث
- ✅ لا يحتاج Python لتشغيله بعد البناء
- ✅ قاعدة البيانات محلية (SQLite)
- ✅ لا يحتاج اتصال بالإنترنت

## توزيع البرنامج:
يمكنك توزيع:
1. ملف `SamirPOS.exe` فقط
2. أو ملف EXE + قاعدة بيانات `pos_system.db` (إذا كانت تحتوي بيانات)

## النسخ الاحتياطي:
**مهم جداً**: احفظ نسخة احتياطية من `pos_system.db` بشكل دوري!
